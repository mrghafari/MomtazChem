✅ ویژگی‌های جدید هدرهای قابل مرتب‌سازی:

  • کلیک روی هدر: هر ستون قابل مرتب‌سازی با کلیک روی عنوان
  • فلش جهت: نمایش فلش ↑ (صعودی) یا ↓ (نزولی) برای ستون فعال
  • هاور افکت: تغییر رنگ به آبی هنگام نگه داشتن موس روی هدر
  • تراز کامل: تمامی هدرها و محتوا کاملاً تراز شده (راست‌چین)
  • عرض ثابت: عرض ثابت برای هر ستون جهت تراز بهتر
  • مرتب‌سازی هوشمند: پشتیبانی از انواع داده (متن، عدد، تاریخ)

🛠️ جزئیات پیاده‌سازی:

  • State Management: sortField (ستون فعال) و sortDirection (جهت مرتب‌سازی)
  • onClick Handler: handleSort() برای تغییر ستون و جهت مرتب‌سازی
  • Sorting Logic: مرتب‌سازی در filteredOrders با پشتیبانی از انواع داده
  • Visual Feedback: نمایش فلش و تغییر رنگ برای ستون فعال
  • Persian RTL: تمامی المان‌ها برای راست‌به‌چپ بهینه شده
  • Responsive: عرض ثابت با حداقل عرض برای تمامی ستون‌ها

📋 ستون‌های قابل مرتب‌سازی:

1. شماره سفارش (orderNumber)
2. مشتری (customerName)
3. مبلغ (totalAmount)
4. وضعیت / بخش (status)
5. کد تحویل (deliveryCode)
6. زمان ثبت سفارش (createdAt)

🎯 هدرهای جدول حالا کاملاً قابل مرتب‌سازی و تراز شده هستند!
